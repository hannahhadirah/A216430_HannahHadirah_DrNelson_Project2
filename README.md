# FreshKeeper - Project 2

## Project Description
FreshKeeper is a mobile application designed to help users manage food items, track expiry dates and reduce food waste. Users can add, edit and delete food items, view detailed information, get recipe suggestions using an API and donate excess food to a community food bank using Firebase.

The app also uses GPS to help users locate nearby food donation centres.

## SDG Theme
SDG 12: Responsible Consumption and Production  
This project focuses on reducing food waste by encouraging better food management and promoting food donation instead of disposal.

## Features
- Add food items with expiry date (Room Database)
- Edit and delete food items
- View detailed food information (status, category, storage tips)
- Get recipe suggestions using TheMealDB API (Retrofit)
- Donate food to Firebase Firestore
- View community food bank (real-time updates)
- Find nearby donation centres using GPS
- Summary dashboard screen

## Tech Stack
- Kotlin
- Jetpack Compose
- Room Database
- Firebase Firestore
- Retrofit API
- Google Maps API
- Fused Location Provider
- Android Studio
